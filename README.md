# Photo Search by Face

Проект для поиска совпадений лиц. Поддерживает:
- CLI-режим (индексация + поиск),
- веб-интерфейс в браузере с массовой загрузкой изображений, автоматической индексацией и ранжированными результатами.

## Установка

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

> `face_recognition` требует системных зависимостей (`dlib`, компилятор и т.д.).

## Веб-интерфейс (рекомендуется)

Запуск:

```bash
python -m app.web
```

Откройте в браузере: `http://localhost:8000`

Сценарий работы:
1. Массово загрузите все фото (.jpg/.jpeg/.png) — индекс построится автоматически.
2. Загрузите фото для поиска и задайте threshold.
3. Получите ранжированный список совпадений (по возрастанию distance).

## CLI

```bash
python -m app.main \
  --images-dir ./photos \
  --query-image ./query.jpg \
  --threshold 0.6 \
  --output ./result.json
```

## Порог похожести

- Меньше `threshold` → строже совпадения.
- Больше `threshold` → мягче совпадения.
- Базовое значение: `0.6`.

## Обработка ошибок

- файл запроса не найден;
- на фото запроса нет лица;
- в каталоге нет валидных изображений;
- на некоторых фото лицо не найдено — логируется warning, обработка продолжается.

## Ограничения

Точность зависит от качества фото, ракурса, освещения и перекрытий лица.
