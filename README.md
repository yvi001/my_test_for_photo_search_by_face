# Photo Search by Face

Небольшой CLI-проект для поиска совпадений лиц: индексирует каталог изображений и ищет фото, похожие на лицо с пользовательского снимка.

## Установка

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

> `face_recognition` требует системных зависимостей (`dlib`, компилятор и т.д.). На некоторых ОС удобнее ставить через готовые колёса/контейнер.

## Запуск CLI

```bash
python -m app.main \
  --images-dir ./photos \
  --query-image ./query.jpg \
  --threshold 0.6 \
  --output ./result.json
```

Аргументы:
- `--images-dir` — каталог с `.jpg/.jpeg/.png` для индексации.
- `--query-image` — путь к фото пользователя.
- `--threshold` — порог расстояния между эмбеддингами (по умолчанию `0.6`).
- `--output` — опциональный путь для JSON-результата (`{"matches": [...]}`).

## Порог похожести

- Меньше значение `threshold` → строже совпадения (меньше ложных срабатываний, но выше риск пропустить нужное лицо).
- Больше значение `threshold` → мягче совпадения (выше полнота, но больше ложных совпадений).
- Типичное стартовое значение: `0.6`, дальше подбирать под конкретный датасет.

## Обработка ошибок

CLI сообщает об ошибках и завершает работу с кодом `1` в случаях:
- файл запроса не найден;
- на фото запроса нет лица;
- в каталоге нет валидных изображений;
- `face_recognition` не установлен.

Если на отдельных фото из каталога лица не обнаружены, это логируется как warning, а обработка продолжается.

## Ограничения

Качество распознавания заметно зависит от:
- качества и разрешения фото;
- ракурса (сильный поворот лица ухудшает результат);
- освещения и теней;
- окклюзий (очки, маска, закрытое лицо).
